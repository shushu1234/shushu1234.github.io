            # 中文题目

            请完成一个函数，输入一个二叉树，该函数输出它的镜像。

            例如输入：

                 4
               /   \
              2     7
             / \   / \
            1   3 6   9
            镜像输出：

                 4
               /   \
              7     2
             / \   / \
            9   6 3   1

             

            示例 1：

            输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]

             

            限制：

            0 <= 节点个数 <= 1000

            注意：本题与主站 226 题相同：https://leetcode-cn.com/problems/invert-binary-tree/

            # 通过代码

            /**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode mirrorTree(TreeNode root) {
        if (root == null) {
            return null;
        }
        TreeNode left = mirrorTree(root.left);
        TreeNode right = mirrorTree(root.right);
        root.left = right;
        root.right = left;
        return root;
    }
}

            # 高赞题解

            二叉树镜像定义： 对于二叉树中任意节点 rootrootroot ，设其左 / 右子节点分别为 left,rightleft, rightleft,right ；则在二叉树的镜像中的对应 rootrootroot 节点，其左 / 右子节点分别为 right,leftright, leftright,left 。

            # 方法一：递归法

            • 根据二叉树镜像的定义，考虑递归遍历（dfs）二叉树，交换每个节点的左 / 右子节点，即可生成二叉树的镜像。
            # 递归解析：
            1. 终止条件： 当节点 rootrootroot 为空时（即越过叶节点），则返回 nullnullnull ；
            2. 递推工作：
              1. 初始化节点 tmptmptmp ，用于暂存 rootrootroot 的左子节点；
              2. 开启递归 右子节点 mirrorTree(root.right)mirrorTree(root.right)mirrorTree(root.right) ，并将返回值作为 rootrootroot 的 左子节点 。
              3. 开启递归 左子节点 mirrorTree(tmp)mirrorTree(tmp)mirrorTree(tmp) ，并将返回值作为 rootrootroot 的 右子节点 。
            3. 返回值： 返回当前节点 rootrootroot ；

            Q： 为何需要暂存 rootrootroot 的左子节点？ A： 在递归右子节点 “root.left=mirrorTree(root.right);root.left = mirrorTree(root.right);root.left=mirrorTree(root.right);” 执行完毕后， root.leftroot.leftroot.left 的值已经发生改变，此时递归左子节点 mirrorTree(root.left)mirrorTree(root.left)mirrorTree(root.left) 则会出问题。

            # 复杂度分析：
            • 时间复杂度 O(N)O(N)O(N) ： 其中 NNN 为二叉树的节点数量，建立二叉树镜像需要遍历树的所有节点，占用 O(N)O(N)O(N) 时间。
            • 空间复杂度 O(N)O(N)O(N) ： 最差情况下（当二叉树退化为链表），递归时系统需使用 O(N)O(N)O(N) 大小的栈空间。

            # 代码：

            Python 利用平行赋值的写法（即 a,b=b,aa, b = b, aa,b=b,a ），可省略暂存操作。其原理是先将等号右侧打包成元组 (b,a)(b,a)(b,a) ，再序列地分给等号左侧的 a,ba, ba,b 序列。

            # 方法二：辅助栈（或队列）

            • 利用栈（或队列）遍历树的所有节点 nodenodenode ，并交换每个 nodenodenode 的左 / 右子节点。
            # 算法流程：
            1. 特例处理： 当 rootrootroot 为空时，直接返回 nullnullnull ；
            2. 初始化： 栈（或队列），本文用栈，并加入根节点 rootrootroot 。
            3. 循环交换： 当栈 stackstackstack 为空时跳出；
              1. 出栈： 记为 nodenodenode ；
              2. 添加子节点： 将 nodenodenode 左和右子节点入栈；
              3. 交换： 交换 nodenodenode 的左 / 右子节点。
            4. 返回值： 返回根节点 rootrootroot 。

            class Solution:
    def mirrorTree(self, root: TreeNode) -> TreeNode:
        if not root: return
        stack = [root]
        while stack:
            node = stack.pop()
            if node.left: stack.append(node.left)
            if node.right: stack.append(node.right)
            node.left, node.right = node.right, node.left
        return root

            class Solution {
    public TreeNode mirrorTree(TreeNode root) {
        if(root == null) return null;
        Stack<TreeNode> stack = new Stack<>() {{ add(root); }};
        while(!stack.isEmpty()) {
            TreeNode node = stack.pop();
            if(node.left != null) stack.add(node.left);
            if(node.right != null) stack.add(node.right);
            TreeNode tmp = node.left;
            node.left = node.right;
            node.right = tmp;
        }
        return root;
    }
}
