            剑指 Offer 66-构建乘积数组(构建乘积数组 LCOF)

            LiuYao 2021-06-26 数组<Array> 前缀和<Prefix Sum>

            # 中文题目

            给定一个数组 A[0,1,…,n-1]，请构建一个数组 B[0,1,…,n-1]，其中 B[i] 的值是数组 A 中除了下标 i 以外的元素的积, 即 B[i]=A[0]×A[1]×…×A[i-1]×A[i+1]×…×A[n-1]。不能使用除法。

            示例:

            输入: [1,2,3,4,5]
输出: [120,60,40,30,24]

            提示：

            • 所有元素乘积之和不会溢出 32 位整数
            • a.length <= 100000

            # 通过代码

            class Solution {
    public int[] constructArr(int[] a) {
        if (a.length == 0) return new int[0];
        int[] b = new int[a.length];
        b[0] = 1;
        int tmp = 1;
        // 计算上三角
        for (int i = 1; i < a.length; i++) {
            b[i] = b[i - 1] * a[i - 1];
        }
        // 计算下三角，tmp为后半段的值
        for (int i = a.length - 2; i >= 0; i--) {
            tmp *= a[i + 1];
            b[i] *= tmp;
        }
        return b;
    }
}

            # 高赞题解

            # 解题思路：

            本题的难点在于 不能使用除法 ，即需要 只用乘法 生成数组 BBB 。根据题目对 B[i]B[i]B[i] 的定义，可列表格，如下图所示。

            根据表格的主对角线（全为 111 ），可将表格分为 上三角 和 下三角 两部分。分别迭代计算下三角和上三角两部分的乘积，即可 不使用除法 就获得结果。

            {:width=500}

            # 算法流程：
            1. 初始化：数组 BBB ，其中 B[0]=1B[0] = 1B[0]=1 ；辅助变量 tmp=1tmp = 1tmp=1 ；
            2. 计算 B[i]B[i]B[i] 的 下三角 各元素的乘积，直接乘入 B[i]B[i]B[i] ；
            3. 计算 B[i]B[i]B[i] 的 上三角 各元素的乘积，记为 tmptmptmp ，并乘入 B[i]B[i]B[i] ；
            4. 返回 BBB 。
            # 复杂度分析：
            • 时间复杂度 O(N)O(N)O(N) ： 其中 NNN 为数组长度，两轮遍历数组 aaa ，使用 O(N)O(N)O(N) 时间。
            • 空间复杂度 O(1)O(1)O(1) ： 变量 tmptmptmp 使用常数大小额外空间（数组 bbb 作为返回值，不计入复杂度考虑）。

            class Solution:
    def constructArr(self, a: List[int]) -> List[int]:
        b, tmp = [1] * len(a), 1
        for i in range(1, len(a)):
            b[i] = b[i - 1] * a[i - 1] # 下三角
        for i in range(len(a) - 2, -1, -1): 
            tmp *= a[i + 1] # 上三角
            b[i] *= tmp # 下三角 * 上三角
        return b
